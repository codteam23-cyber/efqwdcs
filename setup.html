Confirm Settings

Why does Windows flag our software?

Windows Defender uses signature-based detection that flags memory injection techniques as potentially harmful. Game cheats use these same techniques to interact with game processes, which triggers false positives.

Is it safe?

Yes. Our software does not contain any malware, viruses, or harmful code. It simply uses techniques that antivirus software doesn't recognize. You can re-enable Windows protection after installation.
